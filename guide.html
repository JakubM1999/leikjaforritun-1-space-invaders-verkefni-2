LEIÐBEININGAR

Hér munum við byggja herinn og verja jörðina.


ÝTTU Á SPACE TIL AÐ BYRJA

ÁÐUR EN VIÐ BYRJUM

Munið þið eftir Pong? Við munum nota sömu grunnþekkingu hér:


  • Scenes: Við búum til sér scenes fyrir Player, Invader og Bullet.
  • Physics: Við notum Area2D til að skynja árekstra.
  • Scripting: Við notum _process til að hreyfa hluti (notum Script).

Við munum bæta við: Signals (Samskipti) og Groups (Hópstjórnun).

SKREF 1: UPPSETNING (SETUP)

Opnaðu Godot og smelltu á + New hnappinn hægra megin.


  1. Nefndu verkefnið Space Invaders.
  2. Veldu Forward+ sem Renderer (efst til hægri).
  3. Smelltu á Create & Edit.

  4. Farðu í Project -> Project Settings.
  5. Stilltu Window Width á 600 og Height á 800.
  6. Stilltu Stretch Mode á canvas_items.
  7. Búðu til "Main" scene (Node2D) og vistaðu (Ctrl+S) sem main.tscn.

SKREF 2: LEIKMAÐURINN (PLAYER SCENE)

Við byrjum á skipinu okkar.


  1. Búðu til nýtt Scene. Notaðu Area2D sem rót.
  2. Nefndu það Player.
  3. Bættu við Sprite2D (eða ColorRect 40x20 px).
  4. Bættu við CollisionShape2D (RectangleShape) sem passar.
  5. Vistaðu sem player.tscn.

SKREF 3: HREYFING LEIKMANNS

Setjum skriftu (Script) á Player. Við notum clamp til að læsa skipinu inni á skjánum svo það fari ekki út fyrir hliðarnar.

extends Area2D

@export var speed = 400

func _process(delta):
    var input = Input.get_axis("ui_left", "ui_right")
    position.x += input * speed * delta
    
    # Halda innan skjás (20px padding)
    # clamp(gildi, lágmark, hámark)
    position.x = clamp(position.x, 20, 580)

SKREF 4: SKOTIÐ (BULLET SCENE)

Við þurfum að búa til sérstaka senu fyrir skotin svo við getum "instantiate-að" mörg eintök af þeim á meðan leikurinn er í gangi. Þetta verður einfalt en öflugt leysiskot.


  1. Nýtt Scene -> Area2D -> Bullet.
  2. ColorRect (4x15 px, gult).
  3. CollisionShape2D.
  4. VisibleOnScreenNotifier2D (til að eyða því þegar það fer út).

SKREF 5: KÓÐI SKOTSINS

Skotið fer upp og eyðist sjálfkrafa þegar það fer af skjánum til að spara tölvuafl.

extends Area2D
@export var speed = 600

func _process(delta):
    position.y -= speed * delta

func _on_visible_on_screen_notifier_2d_screen_exited():
    queue_free()

Tengdu merkið:

  1. Veldu VisibleOnScreenNotifier2D í senunni.
  2. Farðu í Signals flipann efst (við hliðina á Inspector).
  3. Tvísmelltu á screen_exited().
  4. Smelltu á Connect til að tengja það við skriftuna.

SKREF 6: AÐ SKJÓTA

Nú þurfum við að láta Player vita hvernig skotið lítur út. Við notum @export og PackedScene.


PackedScene: Er eins og "teikning" eða uppskrift af senu sem Godot getur notað til að búa til mörg ný eintök af í einu.

@export var bullet_scene: PackedScene

func _process(delta):
    # ... hreyfing ...
    if Input.is_action_just_pressed("ui_accept"):
        shoot()

func shoot():
    # .instantiate() býr til nýtt eintak eftir teikningunni
    var b = bullet_scene.instantiate()
    get_parent().add_child(b)
    b.position = position

Í Inspector:

  1. Veldu Player node-inn.
  2. Sérðu nýja hlutann "Bullet Scene" í Inspector?
  3. Dragðu bullet.tscn úr FileSystem og slepptu því þangað.

SKREF 7: SKOTHRAÐI (COOLDOWN)

Við viljum ekki að leikmaðurinn geti skotið endalaust ("laser beam"). Við notum Timer til að setja biðtíma milli skota.


  1. Bættu Timer node sem barni á Player.
  2. Veldu Timer-inn og kíktu í Inspector:
    • Hakaðu við One Shot (svo hann ræsist ekki sjálfkrafa aftur).
    • Stilltu Wait Time á 0.5.
func _process(delta):
    # Bætum við skilyrði: Timer verður að vera stopp
    if Input.is_action_just_pressed("ui_accept") and $Timer.is_stopped():
        shoot()
        $Timer.start() # Ræsa klukkuna

SKREF 8: GEIMVERAN (INVADER)

Nú búum við til óvininn. Við notum Groups til að geta stjórnað mörgum geimverum í einu síðar.


  1. Nýtt Scene -> Area2D -> Invader.
  2. Bættu við mynd eða lit (Sprite2D / ColorRect) og CollisionShape2D eins og á Player.
  3. Að bæta í Hóp (Groups):
    • Veldu Invader node-inn.
    • Farðu í Groups flipann efst (við hliðina á Inspector/Signals).
    • Passaðu að valið sé Scene Groups flipinn (það er sjálfgefið).
    • Skrifaðu aliens í boxið og smelltu á Add.

Þessi hópur leyfir okkur að senda skipun eins og "Allir í hópnum 'aliens', hreyfið ykkur!"

SKREF 9: HREYFING GEIMVERU

Geimveran á ekki að hreyfa sig sjálf í hverjum ramma. Í staðinn búum við til fall sem "Main" sena leiksins getur kallað í þegar allur herinn á að taka skref.


Búðu til nýja skriftu (Script) á Invader node-inn:

extends Area2D

# Þetta fall bíður eftir skipun um að hreyfa sig
func move(dx, dy):
    position.x += dx
    position.y += dy

Með því að nota dx og dy getum við notað skipunina call_group síðar til að hreyfa allan herinn samtímis með einni línu af kóða.

SKREF 10: RÖÐUN Í GRIND

Í stað þess að raða 40 geimverum handvirkt, notum við tvöfalda for-lykkju til að búa til fullkomna grind (grid).


  1. Búðu til nýja skriftu (Script) á Main senu leiksins.
  2. Bættu við @export var invader_scene: PackedScene efst.
  3. Mundu að draga invader.tscn úr FileSystem í Inspector-inn á Main node-num!
extends Node2D

@export var invader_scene: PackedScene

func _ready():
    # 5 raðir, 8 dálkar
    for row in range(5):
        for col in range(8):
            var inv = invader_scene.instantiate()
            # Reikna staðsetningu (50px og 40px millibil)
            inv.position = Vector2(60 + col * 50, 50 + row * 40)
            add_child(inv)

SKREF 11: TAKTURINN (TIMER)

Geimveruherinn hreyfist í takt (eins og í upprunalega leiknum). Við notum Timer í Main senu til að stjórna þessu.


  1. Bættu Timer node við Main senu.
  2. Í Inspector: Wait Time: 1.0 og hakið við Autostart: ON.
  3. Farðu í Signals flipann og tengdu timeout() við Main skriftuna.
var direction = 10

func _on_timer_timeout():
    # 'call_group' sendir skilaboð á alla í einu!
    get_tree().call_group("aliens", "move", direction, 0)

Með því að hreyfa hópinn aðeins á hverjum tifi frá Timer, fáum við þetta klassíska arcade hreyfimynstur.

SKREF 12: VEGGJAMERKI (SIGNALS)

Hvernig vitum við ef herinn hittir vegg? Við látum geimveruna láta vita með sínu eigin Signal.


Breyttu Invader skriftunni:

# Skilgreina eigið merki (custom signal)
signal hit_wall

func move(dx, dy):
    position.x += dx
    position.y += dy
    
    # Athuga hvort við séum komin út í kant (600px breidd)
    if position.x > 570 or position.x < 30:
        # Senda merki: "Ég hitti vegg!"
        hit_wall.emit()

Hvað er þetta? Áður notuðum við tilbúin merki (eins og timeout). Hér búum við til okkar eigið merki sem heitir hit_wall svo við getum látið aðra vita nákvæmlega hvenær við viljum.

SKREF 13: TENGJA VEGGJAMERKIÐ

Þar sem við búum til geimverurnar með kóða (í Step 10), þá þurfum við líka að tengja merkin þeirra með kóða.


  1. Opnaðu Main skriftuna.
  2. Inni í for-lykkjunni, bættu við línu sem tengir merkið:
# Inni í _ready() lykkjunni:
var inv = invader_scene.instantiate()
inv.hit_wall.connect(_on_alien_hit_wall) # NÝ LÍNA
add_child(inv)

# Nýtt fall neðst í Main skriftuna:
func _on_alien_hit_wall():
    # Ef einhver hittir vegg, þá snýr allur herinn við og fer niður!
    direction *= -1
    get_tree().call_group("aliens", "move", 0, 20)

Nú höfum við búið til flókið kerfi þar sem hver og ein geimvera fylgist með kantinum, en Main stjórnar heildarhreyfingunni.

SKREF 14: RANDOM ÁRÁSIR

Nú kemur Slembilukka (Randomness) inn. Við viljum að geimverurnar skjóti til baka, en ekki allar í einu.


Í Invader skriftu, bættu þessu við inni í move fallinu:

func move(dx, dy):
    # ... hreyfing ...
    
    # randf() gefur okkur tölu milli 0 og 1
    # 0.05 þýðir 5% líkur á að skjóta í hvert sinn sem herinn hreyfist
    if randf() < 0.05:
        shoot_at_player()

Af hverju hér? Ef við settum þetta í _process myndi geimveran skjóta 60 sinnum á sekúndu! Með því að setja það í move þá gerist það aðeins í takt við taktinn í leiknum.

SKREF 15: ÓVINASKOT

Við þurfum annað skot fyrir óvininn svo hann hitti ekki aðrar geimverur (collision layers) og fari í rétta átt.


  1. Hægri-smelltu á bullet.tscn í FileSystem og veldu Duplicate. Nefndu það enemy_bullet.tscn.
  2. Opnaðu nýju senuna og breyttu litnum á ColorRect í rauðan.
  3. Aðskilja skriftu: Ef við breytum kóðanum núna, þá breytist Player skotið líka!
    • Veldu EnemyBullet node-inn.
    • Í Inspector, finndu hlutann Script.
    • Smelltu á litlu örina (niður) við hliðina á bullet.gd og veldu Make Unique.
# Í enemy_bullet.gd
func _process(delta):
    # Plús (+) þýðir að skotið fer NIÐUR
    position.y += speed * delta

SKREF 16: LEIKMAÐUR DEYR

Við þurfum að vita hvenær óvinaskotið hittir leikmanninn. Við notum Groups til að bera kennsl á hann.


  1. Opnaðu player.tscn.
  2. Veldu Player node-inn, farðu í Groups flipann og bættu honum í hópinn player.
  3. Opnaðu enemy_bullet.tscn og tengdu area_entered() merkið við sína skriftu (Script).
func _on_area_entered(area):
    # Athuga hvort hluturinn sem við hittum sé í 'player' hópnum
    if area.is_in_group("player"):
        area.queue_free() # Drepa leikmann
        queue_free()      # Eyða skotinu
        print("GAME OVER")

SKREF 17: STIGAGJÖF (SCORING)

Til að telja stig þurfum við að láta Main vita þegar geimvera deyr. Við notum annað Signal.


  1. Í Invader skriftunni, bættu þessu við:

# Efst
signal destroyed

# Neðst (mundu að tengja area_entered merkið!)
func _on_area_entered(area):
    if area.is_in_group("bullets"):
        destroyed.emit()
        area.queue_free()
        queue_free()

2. Í Main skriftunni, bættu við stigateljara og tengdu hann í _ready:

# Efst í Main
var score = 0

# Inni í _ready lykkjunni:
inv.destroyed.connect(_on_invader_destroyed)

# Neðst í Main
func _on_invader_destroyed():
    score += 100
    print("Score: ", score)

SKREF 18: SIGUR (WIN)

Hvernig vitum við hvort við höfum unnið? Við teljum einfaldlega hversu margir eru eftir í hópnum aliens.


Í Main skriftunni, bættu þessu við inni í _process:

func _process(delta):
    # Finnum alla í 'aliens' hópnum
    var aliens = get_tree().get_nodes_in_group("aliens")
    
    # Ef listinn er tómur (stærð 0)...
    if aliens.size() == 0:
        print("YOU WIN!")
        # Stöðvum leikinn svo við prentum ekki endalaust
        set_process(false)

Pæling: Gætum við gert þetta á hagkvæmari hátt? Já, við gætum athugað þetta inni í _on_invader_destroyed í staðinn fyrir að gá í hverjum einasta ramma!

SKREF 19: PÚSSNING (VALFRJÁLST)

Leikurinn virkar, en ef þú hefur tíma er gaman að bæta við "safa" (game juice). Hér eru nokkrar hugmyndir:


  • Hraðari her: Geturðu látið $Timer.wait_time minnka í hvert sinn sem geimvera deyr? Þá verður leikurinn erfiðari eftir því sem líður á.
  • Hljóð: Bættu við AudioStreamPlayer2D node og spilaðu hljóð þegar skotið er eða þegar geimvera springur.
  • Bakgrunnur: Farðu í Project Settings -> Rendering -> Environment og breyttu Default Clear Color í dökksvartan.
  • Sprengingar: Geturðu notað CPUParticles2D til að búa til neista?
